Легенда графика

На плоскости заданы два множества точек. Найти все пары окружностей, каждая из которых проходит хотя бы через 3 различные точки одного и того же множества, таких, что касательеая, проведённая к обеим окружностям (внутренняя) параллельна оси ординат.

Легенда графика

  • Красная точка - центр окружности, построенной на точках из первого множества

  • Оранжевая пунктирная окружность - окружность, построенная на точках из первого множества.

  • Голубая вертикальная прямая - внутренняя касательная окружностей, построенных на точках из первого множества и входящих в решение.

  • Светло-зелёная точка - точка из первого множества, которая является одной из точек, образующих окружности, входящие в решение.

  • Синяя точка - центр окружности, построенной на точках из второго множества.

  • Сплошная лиловая окружность - окружность, построенная на точках из второго множества.

  • Тёмно-зелёная вертикальная прямая - внутренняя касательная окружностей, построенных на точках из второго множества и входящих в решение.

  • Розовая точка - точка из второго множества, которая является одной из точек, образующих окружности, входящие в решение.

Решение для нахождения пар окрженостей из одного множества точек:

  • Удаляю дубликаты точек из входного множества.
  • Вычисляю все перестановки без повторения троек точек из полученного множества.
  • Для каждой из этих троек пытаюсь вычислить радиус и координаты центра по следующему алгоритму:
    • Каждая тройка точек является вершинами треугольника, который можно вписать в окружность. Радиусы окружности делят треугольник на 3 равнобедренных треугольника. В каждом из этих треугольников можно найти пару взаимоперпендикулярных векторов. На основании их координат можно записать выражение для нахождения координат центра окружности.
    • Воспользовшись парой таких треугольников, можно получить СЛАУ для нахождения центра окружности. Решаю систему методом Гаусса.
    • Если получилось решить систему, то на основании координат центра вычисляю раудиус окружности.
    • Сохраняю полученные центр окружности, радиус и координаты точек, на которых была построена окружность для дайльнейшего анализа.
  • Перебираю все возможные пары полученных окружностей. Для каждой окружности из пары рассматриваю совпадение абцисс точек, принадлежащих окружностям и имеющих ординаты центров. Если они совпадают, то добавляю окружность в результирующий список.
